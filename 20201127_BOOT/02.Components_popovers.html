彈出提示框 (Popovers)

  1. 除了新增 data-toggle 屬性，還須自行啟動 Javascript
  2. data-placement 彈出顯示方向，預設 top、rignt、bottom、left
  3. data-delay 彈出延遲時間，預設0，單位為毫秒。
  4. title 彈出標題
  5. data-content 彈出內容
  6. data-html="true"，可以在 title 屬性值裡面使用 html 標籤。
  7. data-trigger="click"，預設
  8. data-trigger="hover"，滑鼠滑入
  9. data-trigger="focus"，在畫面上，下一次點擊會關閉
