Sass

Sass 是 CSS 的預處理器，目的是在減少 CSS 反覆重工的動作，以及更有效率維護樣式結構，需進行編譯 (Complier) 成原生的 CSS 才能使用。

Sass 可以使用變數、函式、流程控制、嵌套、繼承(extend)、混合(mixin)等功能，也被稱為 CSS 的程式語言(SassScript)。

Sass 與 Scss

Sass 分為兩種 Sass 與 Scss，兩者同屬 Sass。不同點如下：

副檔名 語法 說明
.sass 縮排語法
沒有 CSS 大括號、結為分號		 語法無法兼容 CSS
.scss 塊語法
寫法與 CSS 一致，兼容 CSS		 語法兼容 CSS

1. 變數(Variables)

$宣告一個變數，其位置是它的作用域。大括號裡面，就是大括號內的區域變數，就像 JavaScript 的 let 宣告

變數可以維持網站的一致性，節省修改成本。

底線(_)和連結線(-)可以互換使用，ex: bg_black 與 bg-black 視為一樣的變數。

2. 流程控制(Flow Control)

@if、@else if、@else

判斷條件可以小括號、也可以不括號。

天氣好冷

@for 迴圈有兩種 through 跟 to

@for $i from (start number) through (end number){...}, 此迴圈包含 end number

@for $i from (start number) to (end number){...}, 此迴不包含 end number

Sass 沒有像 JS 的 for in (js物件), for in, for of，取而代之的是@each

4. Extend(繼承)

繼承可以用 CSS 選擇器 或 % 佔位符來命名

使用 CSS 選擇器當作季持，符合CSS選擇器規範，會被編譯到CSS，%佔位符是屬於SCSS的範疇不會被編譯到 CSS

相同繼承樣式在編譯後會被使用群組選擇器聚集在一起

@extend 沒有參數可以使用

5. Sass的計算

可以做+-*/取餘數

除法比較特別，直接除不會有效果，養成用括號習慣

6. Sass的function

@function 與 @mixin 有點像，這兩差異在於 @function 必須用 @return 返回結果， 他無法再呼叫的地方產生一段CSS樣式

9.SASS 的資料型態

  • 1. number(數字):數字有分有單位或無單位,1,2,1,3,10px ,20rem
  • 2. string(字串):有引號字串"Noto Sans TC",無引號字串 Noto Sans TC
  • 3. color(顏色):blue,#666,rgb(....)
  • 4. boolean(布林值): true false
  • 5. null(空值):null
  • 6. list(列表):用半形空格或逗號作為分割，1.51rem 2rem 0 2rem，或 '', '', ''
  • 7. map(物件):類似 Js 物件，具有 key/value 特性。
