Sass

Sass 是 CSS 的預處理器，目的是在減少 CSS 反覆重工的動作，以及更有效率維護樣式結構，需進行編譯（complier）成原生的CSS才能使用。

Sass 可以使用變數、函式、流程控制、嵌套、繼承(extend)、混合(mixin)等功能，也被稱為 CSS 的程式語言(SassScript)。

sass 與 scss

Sass 分為兩種 sass 與 scss，兩者同屬 Sass。不同點如下:

副檔名 語法 說明
.sass 縮排語法
沒有 css 大括號，結尾分號		 語法無法兼容CSS
.scss 塊語法
寫法與CSS一致，兼容CSS。		 語法兼容CSS

1. 變數(Variables)

$宣告一個變數名稱，其位置是它的作用域，大括號裡面，就是大括號內的區域變數，就像JavaScript的let宣告。

變數可以維持網站的一致性，節省修改成本。

底線(_)和連結線(-)可以互換使用，$bg_black、$bg-black是一樣的變數。

2. 流程控制(Flow Control)

@if、@else if、@else

判斷條件可以小括號，也可以不括號。
